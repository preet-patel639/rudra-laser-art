## ⚡ FASTEST WAY TO GET STARTED (5 Minutes)

### Step 1: Open Terminal/Command Prompt
- **Windows:** Press `Win + R`, type `cmd`, press Enter
- **Mac:** Press `Cmd + Space`, type `terminal`, press Enter

### Step 2: Navigate to This Folder
```bash
cd "d:\Business Website\Rudra Laser Art"
```

### Step 3: Install Everything
```bash
npm install
```
(Wait 2-3 minutes for installation to complete)

### Step 4: Start the Website
```bash
npm start
```

**That's it!** Your website will open automatically in your browser! 🎉

## 🤖 Important: AI Chatbot Setup

The chatbot needs an API key to work with AI:

1. Go to: https://openrouter.ai/
2. Sign up (FREE)
3. Get API key
4. Add to `src/components/Chatbot.js` (line 28)

**Full instructions:** [CHATBOT_SETUP.md](CHATBOT_SETUP.md)

**Note:** The chatbot will still work without API key using basic responses!
